使用说明

                
功能说明：本工具可以识别CSS代码中的颜色值，并直观地显示颜色预览，让您更直观地了解代码中的颜色使用情况。

                
支持的颜色格式：

                
    
                    
  • 十六进制 - 如 #ff0000, #f00
    • 
                    
  • RGB - 如 rgb(255, 0, 0)
    • 
                    
  • RGBA - 如 rgba(255, 0, 0, 0.5)
    • 
                    
  • HSL - 如 hsl(0, 100%, 50%)
    • 
                    
  • HSLA - 如 hsla(0, 100%, 50%, 0.5)
    • 
                    
  • 颜色名称 - 如 red, blue, green
    • 
                    
  • 渐变 - 如 linear-gradient(), radial-gradient()
    • 
                

                
使用步骤：

                
    
                    
  1. 在左侧输入框中输入或粘贴CSS代码
    2. 
                    
  2. 点击"美化代码"按钮
    3. 
                    
  3. 右侧将显示美化后的代码，颜色值旁边会显示颜色预览
    4. 
                    
  4. 下方将显示所有检测到的颜色及其值
    5. 
                    
  5. 点击"复制结果"按钮可以复制美化后的代码
