使用说明

                
功能说明：本工具可以自动调整代码格式和结构，使其更加美观易读。

                
支持的编程语言：

                
    
                    
  • JavaScript - 优化缩进、分号、花括号等
    • 
                    
  • HTML - 优化标签缩进、属性格式等
    • 
                    
  • CSS - 优化选择器、属性、值等格式
    • 
                    
  • Python - 优化缩进、空格、空行等
    • 
                    
  • Java - 优化缩进、花括号、分号等
    • 
                    
  • C++ - 优化缩进、花括号、分号等
    • 
                    
  • PHP - 优化缩进、花括号、分号等
    • 
                    
  • SQL - 优化关键字、缩进、空格等
    • 
                

                
优化选项：

                
    
                    
  • 自动缩进 - 根据代码结构自动添加缩进
    • 
                    
  • 使用空格代替制表符 - 将制表符转换为空格
    • 
                    
  • 统一花括号位置 - 统一花括号的放置位置
    • 
                    
  • 统一引号风格 - 统一使用单引号或双引号
    • 
                    
  • 添加缺失的分号 - 在语句末尾添加缺失的分号
    • 
                    
  • 优化空格和空行 - 移除多余的空格和空行，添加必要的空格
    •
